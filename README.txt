Gif analyser.
(program is still in development)

This program parses GIF files and gives general information about file.

- shows which blocks file contains
- changes pallete 
- saves palette to file
- saves animation frames
- draws animation

Run:
python main.py -h

To run program you need:
- python3
- PIL (pip install pillow)

See more about GIF files format here:
(RU) Short description of the GIF format: http://home.onego.ru/~chiezo/gif.htm#lsd-size
(EN) Specification: https://www.w3.org/Graphics/GIF/spec-gif89a.txt
